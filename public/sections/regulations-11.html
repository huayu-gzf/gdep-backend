广东工程职业技术学院学生创先争优评选实施办法

第一章 总则

第一条 对学生实施创先争优评选的直接目的在于正确评价学生的德、智、体、能表现和综合素质，为学年评优和推荐就业提供依据，实现学生教育管理工作标准化、制度化和科学化。其根本目的在于引导和促进学生德、智、体、能全面协调发展，培养具有创新精神和实践能力的合格人才。

第二条 学生创先争优的内容主要包括五四红旗团总支、红旗学生会、十佳活力团支部、优良学风班集体、十佳社团、百佳团干部、百佳学生干部、优秀共青团员、校园文化先进个人、学生创新奖、十佳大学生、十大自强之星、优秀毕业生等奖项。

第三条 本条例适用的对象为本院全日制学生。

第四条 创先争优评选原则上每学年进行一次，其中，五四红旗团总支、红旗学生会评选每两年进行一次。

第二章 评优的比例和条件

（一）五四红旗团总支

第五条 五四红旗团总支按学校团总支数量的20%评选。五四红旗团总支标兵在连续两次获得红旗团总支的基础上评选。

第六条 确定五四红旗团总支必须具备以下各项条件：

  1. 班子建设好。班子健全，按期换届，民主选举领导班子；定期或经常性地进行干部培训，提高干部的综合素质；班子的整体素质高，凝聚力强；工作制度健全有效，且富有创新意识；密切联系学生和青年教师，全心全意为学生和青年教师服务。
  2. 主题活动好。通过生动活泼、形式多样的主题活动加强大学生思想政治教育，推动学生课外科研，营造优良学风，有效促进学生素质的全面拓展。
  3. 支部建设好。所属支部班子健全，按期换届，民主选举，以身作则，积极引导和服务学生；团支部设置合理，制度健全、落实，能正常地开展工作和活动；认真做好团员教育和管理工作，"推优"工作成效显著，按时完成上级团委交办的任务。
  4. 活动阵地好。建立了党章学习小组等理论学习小组,并能正常开展学习。建有学生社会实践和有关指导服务机构，形成有形的工作平台。

（二）红旗学生会

第七条 红旗学生会按学校二级学院学生会总数的20%评选。确定红旗学生会必须具备以下各项条件:

  1. 思想建设：认真开展学习和贯彻党的路线、方针、政策，积极组织学生会干部开展各种形式的思想教育。
  2. 学风建设：积极配合各二级学院教学工作，在本院营造刻苦勤奋、积极向上的学习氛围；学生会成员表率作用好,在所属班级个人成绩较好。
  3. 组织建设：班子健全，按期换届，民主选举。部门设置合理，队伍稳定，成员精神面貌好，有较强的战斗力、责任心和良好的团队精神。
  4. 宣传工作：积极宣传党、政、团、学的工作，坚持先进的文化方向和正确的舆论导向,积极配合校团委、校学生会开展的各类宣传工作。
  5. 校园文化活动：主动完成学院布置的各项工作，积极组织参与校团委、学生会组织的各类活动并取得较好的成绩，能有效地正确指导下一级学生组织开展工作。

（三）十佳活力团支部

第八条 十佳活力团支部的推荐名额按各学院支部总数的10％计算。评选十佳活力团支部10名，活力团支部提名奖10名。

第九条 确定活力团支部必须具备以下各项条件：

  1. 支委：分工明确，熟悉业务，热心团的工作，积极带领团员开创支部工作的新局面；对本支部每学期的工作有计划、有落实、有总结；在学习、工作、活动开展等方面起表率作用；认真填写好《团支部工作手册》和做好团费收缴情况登记，积极开展团员组织生活和团支部活动。
  2. 集体：在基层团支部建设中起模范带头作用；能结合实际情况开展丰富多彩的第二课堂活动和社会实践活动，收效好；青年志愿者行动、社区援助等社会实践活动常态化并能形成扎实有效的服务项目，建立相对固定的服务基地。团员组织观念强，参与热情高，支部形成积极健康的精神面貌。
  3. 成员：认真学习党和政府新时期的路线、方针和政策，爱国、守法、诚信、知礼，有较强的团员意识，能按时缴纳团费，认真履行团员义务；自觉执行学校的各项规章制度，有良好的班风学风，支部成员的学业总体水平高或有较大进步；能积极配合上级团组织工作，按时完成组织交给的各项任务。
  4. 全体团员本学年没有受任何处分。

（四）优良学风班集体

第十条 优良学风班集体的推荐名额按本院班级总数的10％计算。按照学工部开展的"学风建设活动月"评选结果评定。

第十一条 确定优良学风班集体必须具备以下各项条件：

  1. 思想品德好。全班同学热爱祖国，拥护"一国两制"和国家统一，具有民族自尊心与自豪感，维护祖国尊严和荣誉，遵守社会秩序和社会公德；关心国家大事，认真学习毛泽东思想、邓小平理论以及"三个代表"重要思想，思想品德表现良好，学年内，全班同学思想品德考核成绩合格；具有积极上进、乐于助人、遵纪守法、团结协作、热爱集体、朝气蓬勃、文明健康的良好班风。
  2. 学习好。全班同学学习目的明确，学习态度端正，专业思想稳定，刻苦学习专业文化知识，努力完善自己知识结构，有勤奋、严谨、求实、创新的优良学风，遵守课堂和考场纪律，无考试作弊现象。
  3. 遵纪守法好。遵守国家法律法规法令，遵守高校学生行为准则，学校纪律和各项规章制度，行为规范，举止文明，组织纪律性好，是非观念强，敢于检举揭发坏人坏事，自觉维护校园教学、学习和生活秩序，注意防火、防盗，学年内全班学生没有受违纪处分记录和无因学生个人主观行为而造成的安全事故发生。
  4. 文体卫生好。全班同学积极参加各种有益于身心健康的文体活动，认真上好体育课，坚持体育锻炼；环境意识好，注意校园与宿舍内外环境卫生，积极参加创建"文明宿舍"活动。
  5. 劳动及社会实践好。全班同学热爱劳动，积极参加校园绿化劳动和社会实践活动，并能结合专业特色，认真开展形式多样的科技竞赛活动、社会调查活动，社会公益活动，巩固专业知识，服务社会，锻炼自己，了解国情民情，开阔视野，增长才干，提高素质。
  6. 班干部作风好。班委会成员团结一致，相互协作，能结合班级实际情况制订学年工作计划，求真务实，真抓实干，带领全班同学认真贯彻执行学院各项规章制度和出色完成院、系布置的各项工作任务。班干部要严于律己，以身作则，热心为同学服务，具有无私奉献精神，在广大学生中有较高的威信。
  7. 学生党、团员发挥作用好。学生党员政治立场坚定，方向明确，认真学习邓小平同志建设有中国特色的社会主义理论、"三个代表"重要思想，在德智体等方面起模范带头作用，团支部积极配合学院团委、班委开展争创"先进班集体"、"主题团日"活动，过好团组织生活，做好推优入党工作。团组织的先进性和团员的表率作用突出。

（五）十佳社团

第十二条 十佳社团的推荐没有名额限制，凡符合优秀社团评选条件的均可成为候选社团。

第十三条 确定十佳社团必须具备以下条件：

  1. 接受指导及配合学校开展工作方面：（10分）
    1. 积极接受校团委、学生社团联合会的指导和监督，严格按照《广东工程职业技术学院学生团体管理规定》及其他相关规定开展工作。（6分）
    2. 考勤情况。参照学生团体负责人参与会议考勤记录进行考核，允许学生团体负责人两次授权他人代为出席，超过两次的，从第三次开始视为缺席，（会议包括、换任大会和其他学生社团联合会要求全部团体负责人都需要出席的会议）（4分）
  2. 团体内部建设方面（12分）：
    1. 工作制度化、系统化（3分）
      制定严格、详细的组织章程，建立明确、健全的组织机构，做到制度完备，职责清晰。
    2. 档案材料管理有序（2分）
      建立了完整、全面的会员档案及关于换届的具体情况记录，至少包括会员的姓名、长短号、专业等较为详尽的资料，及时整理会议和开展活动的总结记录，做到计划入档，总结入档。资料齐备，管理规范，设置专门的资料库（袋），建立相应的电子资料库,并有专人管理，无资料不全的情况。
    3. 团体学生干部管理和考核制度健全。（2分）
    4. 会员管理制度健全且实施良好，每学期至少召开一次会员大会，向会员通报团体发展情况以及财务状况。（2分）
    5. 团体上交说明自身内部建设情况的文字材料。(3分)(该文字材料包括自身内部建设的问题与不足、预期建设目标、实际改进方法和改进后结果四个方面，可从本条（一）、（二）、（三）、（四）项的其中任项展开，字数不超过500字)
  3. 团体组织活动方面（32分）：
    团体活动学年度得分(32分)，根据《广东工程职业技术学院学生团体日常活动评价细则》评分。
    计算公式：团体组织活动总分数=团体活动年度得分×32%
  4. 会员满意度评分（16分）
    1. 在每学期末，通过调查部分团体新会员，从而得到对该团体的满意度。
    2. 抽查的新会员人数为团体新会员人数的10％，抽查人数不小于7人且不大于30人，若团体新会员人数不足7人的，则调查该团体全部新会员。
    3. 会员调查主要以问卷或访谈等方式进行。
    4. 在每学年末，通过计算第一学期和第二学期两次满意度调查的平均分，得出团体的最终会员满意度调查分。
  5. 团体财务方面（30分）：
    1. 评分方法：上下两学期，对各个团体进行学期财务审查，得出分数后求两次审查平均得分，以作为该团体当年的财务分数。
    2. 评分根据：《广东工程职业技术学院学生团体财务管理规定》。
  6. 团体处分扣分
    1. （校级）警告、严重警告、暂停活动的，分别扣除10分/次、15分/次、20分/次。
    2. 除上述校级处分外，学生团体联合会给予的其他处分，如通报批评、警告、严重警告等，根据具体情况扣除3—5分/次。（处分结果于全体大会上通报）
    3. 学生团体外联协议合作条款必须如实反映，有隐瞒违规的合作方式，根据具体情况扣除0.5-2分/次。
    4. 学生团体宣传违规，如张贴的海报或其他宣传品没有通过校团委的批准等，每次扣1分。
  7. 团体处分扣分
    1. （校级）警告、严重警告、暂停活动的，分别扣除10分/次、15分/次、20分/次。
    2. 除上述校级处分外，学生团体联合会给予的其他处分，如通报批评、警告、严重警告等，根据具体情况扣除3—5分/次。（处分结果于全体大会上通报）
    3. 学生团体外联协议合作条款必须如实反映，有隐瞒违规的合作方式，根据具体情况扣除0.5-2分/次。
    4. 学生团体宣传违规，如未通过校团委宣传部的审核，而擅自张贴，按每件宣传品扣1分，扣完为止；如通过审核后没有贴在指定的地方，或在非指定的时间内张贴，按每件宣传品扣0.5分，扣完为止；若非上述情况，则根据具体情况再作扣除。
    5. 学生团体迟交必须上交的资料，扣除0.5分/次。学生团体缺交必须上交的资料，扣除1分/次。
  8. 对出现以下情况之一的团体取消评选"十大优秀学生团体"资格
    1. 受到校级严重警告及以上处罚
    2. 六个月内无举办活动

（九）校园文化活动先进个人

第二十条 校园文化活动先进个人的推荐名额原则上各学院为2—3名。

第二十一条 确定校园文化活动先进个人必须具备以下条件：

  1. 思想道德品质和工作、生活作风良好，本年度各类考试无缺考、无重修科目，没有受过任何处分。
  2. 在校园文化活动中表现突出，在同学中有较高的威信，在大型文艺演出或比赛中表现优异者和担任文化文艺类部门或团体主要干部工作扎实、成绩突出者优先推荐。

（十）学生创新奖

第二十二条 学生创新奖的推荐没有名额限制，凡符合学生创新奖评选条件的均可成为候选人。

第二十三条 确定学生创新奖必须具备条件1且符合条件2—12中之一：

  1. 思想政治表现好，热爱祖国，拥护四项基本原则和改革开放，勤奋学习，遵纪守法，没有受到纪律处分。
  2. 在公开发行的学术刊物上发表论文或参与编写出版编著、译著和专著。
  3. 在各种征文竞赛活动中获得省级以上奖励。
  4. 在省级以上科技制作、科技创新竞赛或者各类技能、能力竞赛活动中取得优异成绩或名次。
  5. 参与科研课题研究或科技开发的成果已经通过校级以上鉴定。
  6. 参与的作品获得国家专利。
  7. 在省市级以上体育竞赛（单项）中取得优异成绩。
  8. 在省市级以上文艺汇演或竞赛活动中取得优异成绩。
  9. 在省市级教育厅系统、团学组织评优活动中获奖，被评为优秀共青团干部、优秀共青团员、优秀学生干部等。
  10. 有其他优秀成果或突出成绩者经学校审核通过。

（十一）十佳大学生（感动校园十大年度人物）

第二十四条 确定十佳大学生必须具备以下各项条件：

  1. 参评对象为二年级（含二年级）以上在校大学生。
  2. 模范遵守校纪校规和大学生行为准则，具有良好的道德品质，尊敬师长，诚实守信。
  3. 在校期间学习优异，社会实践、创新能力、综合素质等方面表现突出，无补考现象。
  4. 同等条件下，有优秀事迹者或综合测评中总分和学业表现测评排名在本专业排名靠前的优先考虑。

第二十五条 广东工程职业技术学院十佳大学生（感动校园十大年度人物）包括以下几个类别，并具备以下条件：

  1. 年度敬业人物：用敬畏、敬重的态度对待自己的学习和工作，认真负责、一心一意、精益求精。 在学习上，勤奋好学、追求真知、成绩优秀，在校级以上综合知识竞赛获优异成绩或在著名学术杂志发表学术论文、对某些综合学科做出突出贡献；在社团、学生会等社会组织工作中以及兼职、实习岗位上，勤勤恳恳、尽职尽责、精益求精，业绩突出，为组织、岗位做出重要贡献。
  2. 年度诚信人物：学业诚信，无违反校纪校规、抄袭、旷课、考试作弊、论文抄袭、试验作假、请假造假行为；社会诚信，信守诺言，守时守法，无实践经历或简历造假、通过不正当手段赢得选票、借物不还等行为；经济诚信，无故意拖欠缴费、使用假钞、挪用公款、逃票、骗取奖助学金等行为，并且在信守承诺，恪守诚信方面有先进事迹和突出表现。
  3. 年度友善人物：团结友善，尊师爱生，平易近人，亲和力强，热爱社交，善待身边人，受到家人、师生乃至社会的一致好评，传递和谐友善理念，对人对事有大爱。在助人为乐等体现大爱精神方面起到较强模范作用，并为维护社会和谐做出突出贡献。
  4. 年度爱国人物：热爱祖国，拥护中国共产党及中国特色社会主义，遵守社会道德，热爱中华民族优秀传统文化，并付出相应的实际行动。在校期间能勤于学习、努力工作、甘于奉献，关注民生，心系社会，具有较强的社会责任感和使命感，有坚定的理想信念，通过各种方式为祖国的和平统一、繁荣富强、社会和谐，以及为中华民族优秀文化传承发展做出一定贡献。
  5. 年度自强人物：面对贫困、疾患、残障、自然灾难等各种艰难境遇或挫折，能够不畏艰难、自信乐观 、积极应对，通过自己的努力在逆境中创造奇迹。
  6. 年度志愿人物：奉献自我，服务社会，不求名利，积极帮助身边的弱势人群，积极为推动人类发展和社会进步的社会公共事业提供志愿服务，在各级志愿工作中表现出色。
  7. 年度科技人物：在科学技术研究方面通过自身刻苦钻研，克服科研历程的层层险阻，最终取得优秀成果，其科研成果对所在研究领域，有巨大深远的影响。
  8. 年度艺术人物：在高水平艺术大赛中取得优异成绩，在艺术创作、艺术评论、艺术演出等方面做出较大贡献，其作品、事迹在校园、社会上有较大的影响力。
  9. 年度体育人物：热爱体育运动并时刻坚持奥林匹克精神。在体育竞技方面通过自身刻苦，勤于练习，在省级或国家级比赛上获表彰，并对体育活动以及体育文化传播工作做出贡献。
  10. 年度创业人物：对大中专学生自主创业有较为突出的能力，以及结合自身专业特点积极开展自主创业活动，并创造一定社会价值，取得良好的社会效益和经济效益。

第二十六条 以上条件中，候选人如有参加与该项人物相关的活动则需附上相关书面证明,如有获奖则附上证书复印件。

（十二）十大自强之星

第二十七条 确定十大自强之星学生必须具备以下各项条件：

  1. 参评学生须是我校在校品学兼优的家庭经济困难学生，上届活动中已获评学生不再参评。
  2. 具有良好的思想政治素质，综合表现优异（学年综合素质测评须在专业前10%），品行端正，热心公益，乐观向上。学年无补考、重修现象，无旷课等违纪记录。
  3. 勇于克服家庭经济、身体疾病等方面的困难，具有吃苦耐劳和拼搏奉献的精神，在压力、挫折和磨难面前表现出顽强的毅力，坚强的意志，在学生中能起到模范带头作用。
  4. 积极参加公益志愿者服务及社会实践活动，有自立自强的表现。
  5. 有突出个人先进事迹、个人的经历或行为事迹感人，体现良好的道德品质和时代精神。
  6. 提交材料必须真实有效。

（十三）优秀毕业生

第二十八条 优秀毕业生评选根据《学生素质测评实施细则》相关规定评定。

第三章 评优的程序

第二十九条 学生"创先争优"的评选按如下程序进行：

  1. 填写推荐表。
  2. 撰写评优申报材料。申报的集体或个人需另附介绍材料，字数控制在2000字左右。评优材料要突出先进性和典型性，其中"五四红旗团总支" 、"红旗学生会"的申报材料需介绍该组织近两个学年来的工作情况，其它申报材料只需介绍自近一学年来的情况。
  3. 二级学院审查。各学院的评优推荐工作由学院团总支或学生会统一组织申报集体和个人填写相应表格，经二级学院领导审查，签署意见并加盖公章后上报校团委；校级各学生机构的评优推荐工作由校团委统一组织。
  4. 校团委审核。校团委对上报的材料进行评选、审核，并报学校主管领导审批。
  5. 表彰。学校主管领导审批后，印发文件，对评选为先进集体和个人的单位及学生予以表彰和奖励。

第四章 学生创先争优各项奖项奖励办法

第三十条先进集体奖励办法：

  1. "五四红旗团总支"、"红旗学生会"颁发奖牌，奖励奖金一千元。
  2. "十佳活力团支部"奖励奖金200元，"十佳活力团支部提名奖"奖励奖金100元。
  3. "优良学风班集体"、"优秀社团"颁发奖状。

第三十一条 优秀个人奖励标准

  1. "优秀团干"、"优秀学生干部"、"优秀共青团员"、"校园文化先进个人"、"优秀毕业生"颁发荣誉证书。
  2. "学生创新奖"个人颁发荣誉证书,按市、省、国家三级给予三百、五百、一千元奖励。获奖团队颁发荣誉证书,按市、省、国家三级给予一千、二千、三千元奖励奖金。
  3. "十佳大学生"颁发荣誉证书，奖励奖金3000元。"十佳大学生"提名奖颁发荣誉证书和奖金1000元。
  4. "十大自强之星"颁发荣誉证书和奖金3000元；"十大自强之星"提名奖颁发荣誉证书和奖金1000元。
